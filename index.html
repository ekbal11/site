What is Web Design?

Web design refers to the design of websites that are displayed on the internet. Web design is a multi-disciplinary job, where you'd need not only knowledge in design but also skills in developing a website (HTML, CSS, JavaScript). Some web designers are also involved in interaction design.

What Is Web development?

Web development is the building and maintenance of websites; it's the work that happens behind the scenes to make a website look great, work fast and perform well with a seamless user experience. Web developers, or 'devs', do this by using a variety of coding languages
